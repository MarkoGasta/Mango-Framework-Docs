## About Features

Theres tons of useful features in **Mango Framework**. All of them have, at one time, been used in practice. Features are constantly developing and improving. Suggestions for additional features are open.

## Behavior Features
- One-script runtime.
- Sequential module loading.
- Server loading priority (server always loads fully first)

## Module Management Features
- Both server and client sided module loading.
- Dependency and priority management.
- Caching of modules to a quickly accessible location.
- Handling of modules with unexpected failures.
- Prevention of cyclic dependencies.
- Infinite modular expandability.
- Extensive debugging information.

## Organization Features
- Semantic module storage.
- Proprietary standardized layout.

## Data Accessibility Features
- Faster global value storage.
- Per-player cache management.
