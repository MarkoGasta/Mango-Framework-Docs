## What is it?

**Mango Framework** is my take on creating a professional and feature full, but still easy to use, framework to handle all scripts dor creating experiences on *Roblox*. While it might be similar to others, this framework has it's own style. The layout in the framework is straight forward and easy to learn. It's been revised over multiple years to handle anything you might need in a game. Depending on the branch you choose, it's functionality can range from only the essentials to a full set of integrated tools for speeding up development.
